# CV PDF Placeholder

Put your CV/resume PDF file here and name it `cv.pdf`.

After placing your `cv.pdf` in this directory:
1. The Download CV button on the `/cv` page will link to `/cv.pdf`.
2. Users will be able to download your CV directly.
3. No code changes are needed — just place the file.

## Steps
1. Export your CV as PDF.
2. Rename it to `cv.pdf`.
3. Put it in the `public/` folder (same directory as this file).
4. Deploy — the link will work automatically.
